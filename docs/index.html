Задание 1 – Крестики-нолики

Определите победу крестиков, для каждой завершённой партии в крестики-нолики. Крестики ходят первыми.

Файл с данными Заготовка для кода на Python

Описание входных данных

Каждая строка входного файла описывает одну партию. В строке описываются последовательно все клетки (слева направо, сверху вниз). В каждой клетке может быть крестик (x), нолик (o) или клетка может быть пустой (b); Порядок значений в файле:
  1. top-left-square
  2. top-middle-square
  3. top-right-square
  4. middle-left-square
  5. middle-middle-square
  6. middle-right-square
  7. bottom-left-square
  8. bottom-middle-square
  9. bottom-right-square

Описание выходных данных

В ответ предоставите алгоритм и выходной файл с итогами каждой партии. В каждой строки файла с итогами партий должна быть отмечена либо победа (1) либо проигрыш или ничья (оба последних варианта обозначаются так: 0) для крестиков.

Порядок итогов в выходном файле должен соответствовать порядку партий, описанному во входном файле.
